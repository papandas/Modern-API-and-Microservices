# Prerequisite

1. Nodejs
2. Docker

# Install gRPC tools globally

`npm install -g grpc-tools`

# Install nodemon globally

`npm install -g nodemon`

# Clone the repository

`git clone [Link_To_Repository]`

# Update node modules

`npm install`

# Generate protobuf package

`npm run generate:dummy`

# Run server

`npm run server`

# Run Client

`npm run client`
